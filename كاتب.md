# كاتب

**كاتب** تطبيق ويب عربي بسيط يساعد على تصحيح وتحسين وتبسيط النصوص العربية باستعمال Claude API عبر مسار خادم آمن في Vercel.

## الملفات

| الملف | الوظيفة |
|---|---|
| `index.html` | واجهة التطبيق الرئيسية. |
| `api/katib.js` | مسار API آمن يستدعي Claude من جهة الخادم. |
| `vercel.json` | إعدادات بسيطة للنشر على Vercel. |
| `README.md` | شرح المشروع وطريقة التشغيل. |

## طريقة الرفع إلى GitHub

ارفع الملفات إلى المستودع مع الحفاظ على الأسماء كما هي بالضبط:

```txt
index.html
README.md
vercel.json
api/katib.js
```

إذا كنت ترفع من الهاتف، تأكد ألا تصبح الأسماء مثل `index (1).html` أو `vercel (2).json`.

## إعداد Vercel

بعد ربط GitHub مع Vercel، افتح إعدادات المشروع في Vercel ثم أضف متغير البيئة التالي:

```txt
ANTHROPIC_API_KEY
```

ضع قيمة مفتاح Claude داخل هذا المتغير فقط. لا تضع المفتاح داخل `index.html` أو داخل أي ملف ظاهر في GitHub.

بعد إضافة المتغير، أعد النشر من Vercel عبر **Redeploy**.

## ملاحظة أمنية

إذا ظهر مفتاح Claude في صورة أو ملف أو محادثة، اعتبره مكشوفًا. الأفضل تعطيله من منصة Claude وإنشاء مفتاح جديد ثم وضعه في Vercel Environment Variables.

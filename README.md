# كاتب — أداة تحسين الكتابة العربية

أداة ذكية تصحح وتحسن الكتابة العربية مدعومة بـ Claude AI.

## خطوات النشر على Vercel

### 1. رفع الملفات على GitHub
1. افتح github.com وسجل دخولك
2. اضغط "New repository"
3. اسمه: `katib`
4. اضغط "Create repository"
5. ارفع ملفي `index.html` و `vercel.json`

### 2. ربط GitHub بـ Vercel
1. افتح vercel.com وسجل دخولك
2. اضغط "Add New Project"
3. اختر مستودع `katib`
4. اضغط "Deploy"

### 3. إضافة API Key بأمان
1. في لوحة Vercel اذهب إلى Settings → Environment Variables
2. أضف متغير جديد:
   - Name: `ANTHROPIC_API_KEY`
   - Value: مفتاحك الجديد (sk-ant-...)
3. اضغط Save وأعد النشر (Redeploy)
